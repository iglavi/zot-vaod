# הורדה יומית אוטומטית של פסקי דין — מדריך התקנה ל-Windows

מדריך זה מסביר, צעד אחר צעד, איך להקים על מחשב/שרת Windows הורדה יומית
אוטומטית של פסקי הדין ממאגר הרשות השופטת (`decisions.court.gov.il`), כך
שבכל בוקר יורדים כל פסקי הדין החדשים (Word + PDF) והמאגר מתעדכן לבד.

> אין צורך בידע טכני מוקדם — פשוט בצעו את השלבים לפי הסדר.

---

## שלב 1 — התקנת Python (פעם אחת)

1. היכנסו ל־**https://www.python.org/downloads/** ולחצו **Download Python**.
2. הריצו את קובץ ההתקנה. **חשוב מאוד:** בתחתית המסך הראשון, סמנו ✅ את
   **"Add python.exe to PATH"**, ורק אז לחצו **Install Now**.
3. בסיום, כדי לוודא: פתחו **Command Prompt** (חפשו "cmd" בתפריט התחל),
   הקלידו `python --version` ולחצו Enter. אמור להופיע מספר גרסה (למשל 3.12).

## שלב 2 — הורדת הפרויקט למחשב (פעם אחת)

הדרך הפשוטה:
1. היכנסו ל־**https://github.com/iglavi/zot-vaod**.
2. לחצו על הכפתור הירוק **Code** → **Download ZIP**.
3. חלצו את ה-ZIP לתיקייה נוחה, למשל `C:\zot-vaod`.

(אם מותקן git, אפשר במקום זאת: `git clone https://github.com/iglavi/zot-vaod.git`)

## שלב 3 — התקנת הרכיבים (פעם אחת)

1. פתחו **Command Prompt**.
2. עברו לתיקיית הפרויקט — הקלידו (התאימו לנתיב שלכם):
   ```
   cd C:\zot-vaod
   ```
3. התקינו את החבילות:
   ```
   pip install -r requirements.txt
   ```
   (ההתקנה לוקחת דקה-שתיים.)

## שלב 4 — הגדרת פרטי הגישה (פעם אחת)

1. בתיקיית הפרויקט יש קובץ בשם **`.env.example`**. העתיקו אותו וקראו לעותק
   **`.env`** (בדיוק כך, עם הנקודה בהתחלה ובלי סיומת).
   - דרך מהירה ב-Command Prompt: `copy .env.example .env`
2. פתחו את `.env` ב-Notepad ומלאו:
   ```
   DECISIONS_USER=User106G
   DECISIONS_PASSWORD=הסיסמה-שלכם
   ```
   שמרו וסגרו.

> 🔒 הקובץ `.env` נשאר רק אצלכם על המחשב ולעולם לא נשלח לאינטרנט/GitHub.
> מומלץ להחליף את הסיסמה מול הרשות השופטת אם היא נחשפה, ולעדכן כאן בלבד.

## שלב 5 — הרצה ראשונה (בדיקה)

הריצו ידנית פעם אחת כדי לוודא שהכול עובד:
```
python fetch_daily.py
```
- ההרצה הראשונה מורידה את **כל** תיקיות התאריך שנמצאות בשרת (עד ~30 יום),
  ולכן עשויה לקחת זמן ולהוריד הרבה קבצים. זה תקין.
- בסוף אמורה להופיע שורת סיכום: כמה קבצים ירדו, וכמה רשומות יש במאגר.
- אם מופיעה שגיאת התחברות — בדקו שם משתמש/סיסמה ב-`.env`
  (שימו לב לאותיות גדולות/קטנות).

לאחר מכן אפשר להריץ את אפליקציית החיפוש:
```
python -m streamlit run app.py
```

## שלב 6 — תזמון אוטומטי כל בוקר (פעם אחת)

כדי שההורדה תרוץ לבד כל בוקר, נגדיר משימה מתוזמנת. בתיקייה כבר יש קובץ מוכן
בשם **`run_daily.bat`** שעושה את הכול.

**הדרך הקלה (פקודה אחת):** פתחו **Command Prompt כמנהל** (לחיצה ימנית על cmd
→ "Run as administrator"), והריצו (התאימו את הנתיב לתיקייה שלכם):
```
schtasks /Create /SC DAILY /ST 07:00 /TN "GiluyNaot-Daily" /TR "C:\zot-vaod\run_daily.bat" /F
```
זהו — המשימה תרוץ כל יום ב-07:00 בבוקר. אפשר לשנות את השעה (`/ST 07:00`).

**לבדיקה מיידית** שהמשימה עובדת:
```
schtasks /Run /TN "GiluyNaot-Daily"
```
מה שירד יירשם בקובץ `daily_log.txt` בתיקיית הפרויקט.

> **חלופה דרך ממשק:** פתחו **Task Scheduler** → **Create Basic Task** →
> שם: `GiluyNaot-Daily` → Trigger: **Daily** בשעה הרצויה → Action:
> **Start a program** → Program: הנתיב ל-`run_daily.bat` → Finish.

## איך זה עובד (בקצרה)

בכל הרצה, `fetch_daily.py`:
1. מתחבר לשרת עם שם המשתמש והסיסמה (HTTP Basic).
2. סורק את תיקיות התאריך, ומוריד רק קבצים **חדשים** (מדלג על מה שכבר יש)
   לתוך `documents\<תאריך>\`.
3. מריץ בנייה מחדש של האינדקס — כולל **חילוץ אוטומטי** של בית המשפט, מספר
   התיק, הצדדים, השופט/ת וסוג ההחלטה מתוך גוף פסק הדין (כי בקבצים אין CSV).
4. המאגר מתעדכן, ואפליקציית החיפוש (`streamlit run app.py`) מציגה מיד את
   פסקי הדין החדשים.

הקבצים **נצברים** אצלכם (גם אחרי שהשרת מוחק תיקיות ישנות), כך שנבנה לאורך זמן
ארכיון מלא לחיפוש — הרבה מעבר ל-30 הימים שבשרת.

## הוספת פסקי דין בעצמכם

רוצים להוסיף פסק דין שלא ירד אוטומטית? פשוט גררו את קובץ ה-PDF/Word לתוך
התיקייה `documents\manual\`. בהרצה היומית הבאה (או בהרצה ידנית של
`python fetch_daily.py`) הוא ייקלט אוטומטית — פרטי התיק (בית משפט, מספר
תיק, שופט/ת וכו') יחולצו מתוך הקובץ עצמו, בדיוק כמו בהורדות הרגילות.

## אחסון קבצי המקור באתר הציבורי (אופציונלי)

כברירת מחדל, האתר הציבורי מציג רק את **הטקסט** של פסקי הדין. כדי שיהיה
אפשר גם להוריד את **הקובץ המקורי** (PDF/Word) מהאתר, אפשר לחבר אחסון חינמי
של Cloudflare R2 — פרטים ב-`DEPLOY.md`.

## פתרון תקלות

| תופעה | פתרון |
|---|---|
| `python` לא מזוהה | לא סומן "Add to PATH" בהתקנה — התקינו מחדש עם הסימון. |
| שגיאת התחברות (401) | בדקו `DECISIONS_USER`/`DECISIONS_PASSWORD` ב-`.env` (אותיות גדולות/קטנות). |
| המשימה לא רצה | ודאו שהנתיב ב-`schtasks` מדויק, ושהמחשב דלוק בשעה שנקבעה. |
| רוצים רק את היומיים האחרונים | הוסיפו ל-`.env` את השורה `DECISIONS_DAYS=2`. |
